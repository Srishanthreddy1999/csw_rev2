# Check Sheet Portal Prototype

This folder contains the basic front-end prototype ready to upload to GitHub.

## Files to upload

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `sample-checkpoints.csv`

## Demo login

- Admin: `ADMIN001` / `admin123`
- Employee: `EMP001` / `emp123`

## Notes

- This prototype can be configured to use Firebase `Firestore` for remote storage
- Excel reading is handled through CDN-based `xlsx`
- PDF download is handled through CDN-based `jsPDF`
- Add your Firebase config in `index.html` and the app will persist data remotely
